## 数和数的运算

### 整数

用来表示物体个数的 1, 2, 3, ...叫做自然数，一个物体也没有，用 0 表示

自然数和 0 都是整数

一（个）、十、百、千、万、十万、百万、千万、亿...都是计数单位，其中“一”是最计数的基本单位，每两个相邻的单位之间的进率都是 10，这样的计数法叫做十进制计数法

数位指一个数中每一个数字所占的位置
