正则表达式在通过一定的搜索模式下进行一个或多个匹配（即 ASCII 或 unicode 字符的特定序列）从任何文本中提取信息时非常有用

应用领域从验证到解析/替换字符串，将数据转换为其他格式以及网络爬虫

一旦学会了正则，几乎可以在所有编程语言中使用

## ^ 和 $

+ `^start` - 匹配任意以`start`开头的字符串
+ `end$` - 匹配任意以`end`结尾的字符串
+ `^JQiue$` - 匹配`JQiue`
+ `the` - 匹配含有`the`的字符串
