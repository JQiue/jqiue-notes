## 编程语言的选择

+ C：非面向对象，写法复杂，大量内存管理代码，性能极高
+ C++：写法复杂，大量的内存管理代码，性能极高
+ JavaScript、Python：依赖解析器，同一个逻辑使用不同的写法会影响性能
+ Java：语法丰富严谨，完全面向对象，写法难度低，中规中矩

数据结构与算法是和编程语言无关的，使用自己最熟悉的语言实现一遍，本系列会分别使用多种语言去实现
