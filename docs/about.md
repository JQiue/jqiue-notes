## 内容和排版

本站记录的是我自大学以来学习的各类知识，想写（抄）的很多，总结是一种很花精力的事情

- [x] 10%（2021.3.21）
- [x] 20%（2021.10.28）
- [x] 25%（2022.3.22）
- [x] 35%（2023.8.8）
- [ ] 50%

本站统一中文文案，排版的相关用法，降低访问者的沟通成本，并约束所有的文档

+ 空格
  + 中英文之间添加空格
  + 中文和数字之间添加空格
  + 数字和单位之间不添加空格
  + 全角标点和其他字符之间不加空格
+ 全角和半角
  + 使用全角中文标点
  + 数字使用半角字符
  + 完整的英文整句、特殊名词、在其内容中使用半角
